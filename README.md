# nft
NFT Creation

## Environment
ERC721, web3.js, ether.js, IPFS, Hardhat, OpenZeppelin, Alchemy, 

## Installation
npm install --save-dev hardhat

## Create Project
npx hardhat

## Dependencies
npm i @openzeppelin/contracts --save

## Compile Smart Contracts
npx hardhat compile

## Run Tests
npx hardhat test
